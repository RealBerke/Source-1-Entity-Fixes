# Entity Fixes
**Usage:** Download the **entity_fixes.nut** file, place it in your game's **scripts/vscripts/** folder and type **entity_fixes** in the **Entity Scripts** field in entity's properties.
## func_movelinear
Fixes movement when parented
## func_rotating
Fixes rotation limit.
## point_teleport
Fixes destination not updating when moved.
## trigger_push
Fixes direction not being relative to spawn angle.
